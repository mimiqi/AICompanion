# 2026-04-22 10:59 - 静态立绘 PNG 修复 (webp → png)

## 一、本次会话目标

解决桌宠区域黑屏、利托里奥 SD 立绘无法显示的问题。经过系统性排查最终定位到 OLV 服务器的文件类型白名单限制 `.webp` 格式返回 403。改用用户提供的 PNG 版本一举修复。

会话起点：图片文件 `AzurLane_Littorio.webp` 已放入 `avatars/`,`model_dict.json` 指向了该文件,但浏览器始终显示黑屏(无立绘)。

会话终点：PNG 图片正确显示,`/avatars/AzurLane_Littorio.png` HTTP 200 返回 39 KB。

---

## 二、根本原因

[Open-LLM-VTuber/src/open_llm_vtuber/server.py](../../Open-LLM-VTuber/src/open_llm_vtuber/server.py) 第 44-52 行定义了 `AvatarStaticFiles` 类:

```python
class AvatarStaticFiles(CORSStaticFiles):
    allowed_extensions = (".jpg", ".jpeg", ".png", ".gif", ".svg")
    if not any(path.lower().endswith(ext) for ext in allowed_extensions):
        return Response("Forbidden file type", status_code=403)
```

`.webp` 不在白名单,OLV 对每次 `/avatars/AzurLane_Littorio.webp` 请求都返回 `403 Forbidden file type`。前端 `<img src="/avatars/...">` 加载失败,图片不渲染,桌宠区域显示纯黑。
