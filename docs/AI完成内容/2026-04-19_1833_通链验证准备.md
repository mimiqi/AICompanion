`gpt_sovits_tts` 整块未删,后续训完模型只回改 `tts_model` 一行即可。

### 3.2 companion_panels.py — 路径 bug 修复

```python
# 修改前 (parents[3] 跳到 CODE_PYTHON,_MCP_SERVERS_PATH = CODE_PYTHON/mcp_servers,不存在)
_PROJECT_ROOT = Path(__file__).resolve().parents[3]
_MCP_SERVERS_PATH = _PROJECT_ROOT.parent / "mcp_servers"

# 修改后 (语义清晰,parents[2] 是 OLV 根，再 .parent 才是仓库根)
_OLV_ROOT = Path(__file__).resolve().parents[2]
_REPO_ROOT = _OLV_ROOT.parent
_MCP_SERVERS_PATH = _REPO_ROOT / "mcp_servers"
```

修复前 `/api/companion/health` 会两个 panel 都报 503;修复后 `{"ok":true,"todo_enabled":true,"mail_enabled":true}`。

### 3.3 .venv Python 版本偏离记录

Plan 原指定 Python 3.12,实际落到 3.10.20。原因链:
1. `Open-LLM-VTuber/.python-version` 内容是 `3.10`(项目级固定)
2. `pyproject.toml` `requires-python = ">=3.10,<3.13"`
3. `uv add faster-whisper` 重新解析时按项目固定优先,选了已存在的 3.10.20

**结论：3.10.20 在 OLV 支持范围内,且 Windows 上 torch/faster-whisper 的 3.10 wheel 兼容性反而更好**,不强行回切以避免重新解包近 3GB。如以后想统一到 3.12,只需 `echo 3.12 > .python-version && uv sync --reinstall`。
