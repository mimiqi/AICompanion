## 四、关键技术决策与坑

### 4.1 数据源 — 为什么放弃萌娘百科

| 候选 | 状态 |
| --- | --- |
| zh.moegirl.org.cn 直接 GET | HTTP 403 Access Denied |
| zh.moegirl.org.cn `/api.php?action=parse` | `action-notallowed: Unauthorized API call` |
| mzh.moegirl.org.cn 镜像 API | 同样 Unauthorized |
| moegirl.org / en.moegirl.org.cn | 301 跳转 / 504 超时 |
| **wiki.biligame.com/blhx** | **HTTP 200, HTML 规整, 中文台词与日语 CV 配对** |
| azurlane.koumakan.jp | 200 但是英文翻译,与中文台词不配对 |

bilibili wiki 是 BWiki 旗下的官方游戏数据库,对游戏数据的引用合规;CDN `patchwiki.biligame.com` 也无反爬。

### 4.2 解析坑 — `table-ShipWordsTable` 的 class 名被滥用

最初 selector `table.table-ShipWordsTable` 抓出 6 个 table,但其中**第一个其实是"情人节礼物"H2 下的特殊场景表**(class 复用)。它没有 `tr[data-key]` 行,parse 出 0 records,但占了我"通常"皮肤的位置,导致**所有皮肤标签整体后移一格**(通常→换装1, 换装1→换装2, ..., 改造→雨中的等候, 誓约→无)。

修复后的 `discover_skin_tables()` 用 3 个真实信号代替 tab 列表配对:

1. 必须含 `tr[data-key] p.ship_word_line[data-lang='zh']` — 排除"情人节礼物"这种纯文本表
2. 有 `data-title` 属性 → 皮肤名直接用 data-title(`那不勒斯之光` / `Calabria Aurea` / `居家私人时间` / `雨中的等候`)
3. 无 data-title 但符合条件 1 → 即"通常"皮肤(BWiki 通常皮肤的表无 data-title)

### 4.3 PowerShell 控制台中文乱码 ≠ 文件乱码

PowerShell 默认 GBK 输出,展示 UTF-8 文件名时全显示成乱码。**实际文件名/内容写到 NTFS 是正确的 UTF-8**(NTFS 文件名是 Unicode 不依赖代码页)。验证用 Python `read_text(encoding='utf-8')` 即可看到真中文。这点在写日志时记录,免得用户自己 ls 一眼以为脏数据。

### 4.4 皮肤数 7 vs 5

Plan 写"7 个皮肤 tab"(通常/换装1~4/改造/誓约),实际只有 **5 个有独立 voice 表**。原因:
- "改造"皮肤:利托里奥目前没有改造立绘,这个 tab 在页面上是空的
- "誓约"皮肤:誓约换装在 BWiki 上不另起 voice 表(共用通常皮肤台词,只有立绘换)

不影响训练 — 5 套语音、5 种音色变化、52 段对话已经足够覆盖 SoVITS-V2 所需的 5 分钟以上素材。

## 六、用户后续要做的事

1. **打开几个 .mp3 听一下** — 确认音质 OK(BWiki 上是 mp3 压缩,128 kbps 左右,SoVITS-V2 训练可用但建议训练前转 wav 16k)
2. **在 SoVITS-WebUI 里走预处理流程**(参见 [docs/sovits_setup.md](../sovits_setup.md)):
   - Tab1 一键三连:语音切片(虽然时长合适但建议过 slicer 兜底) → ASR 重新生成更精确文本(可选,因为我们已有 BWiki 文本) → 文本校对
   - Tab2 训练:先 SoVITS 模型 ~30 epoch,再 GPT 模型 ~15 epoch(具体 epoch 看 loss 曲线)
3. **训出模型后**,把 ref_audio 路径回填到 [Open-LLM-VTuber/conf.yaml](../../Open-LLM-VTuber/conf.yaml) 第 363/365 行:
   ```yaml
   gpt_sovits_tts:
     ref_audio_path: 'D:/GPT-SoVITS/training_source/littorio/00_通常_login_等待虽然无.mp3'  # 选一个 3-10s 的清晰短句
     prompt_text: '等待虽然无趣,但等待你并不无趣,指挥官'
   ```
   再把 `tts_model` 从 `edge_tts` 切回 `gpt_sovits_tts`
